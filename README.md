# 🎉 Event Management HTML Project

A simple and attractive **Event Management Website** created using **HTML**. This project displays upcoming events, event images, a fashion store link, and contact information in a clean and easy-to-understand layout.

## 📖 About the Project

This project is a beginner-friendly static website built as part of learning HTML. It demonstrates the use of basic HTML elements such as headings, paragraphs, images, hyperlinks, horizontal rules, and text formatting to create an event information webpage.

## ✨ Features

* 📅 Upcoming events section
* 🖼️ Event image gallery
* 👗 Fashion store link (Myntra)
* 📞 Contact information section
* 🎨 Dark-themed webpage
* 📝 Simple and easy-to-read layout

## 🛠️ Technologies Used

* HTML5

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Mithraa-N/event-management-html.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

> **Note:** If your images are not displaying, replace the local image paths such as:

```html
C:\Users\HP\Desktop\...\img\event1.png
```

with relative paths:

```html
img/event1.png
```

and

```html
img/event2.png
```

This ensures the project works correctly on GitHub and on other computers.
